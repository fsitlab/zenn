# sections/ — 記事用MD置き場

本番に載せる原稿の最小単位（セクション）を、1セクション＝1MDで置く場所。

## 命名規約（予定）
`<カテゴリ記号><連番>-<スラッグ>.md`
例: `C07-deck.md`, `R13-metaphor-compare.md`

カテゴリ記号は `work/brainstorm-sections.md` の A〜R に対応。
並べ替えても追跡できるよう、記号＋連番は固定し、順序は別途「目次案」で管理する。
