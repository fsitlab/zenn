# tasks/ — タスク管理

本づくりのToDo・進捗を管理する場所。

## 運用（案）
- `todo.md` … 直近のやることリスト（チェックボックス）
- フェーズ管理：発散 → 収束（目次案） → 執筆 → レビュー → ファクトチェック → 本番移行
- 大きめの単位は見出しで、細かいタスクは `- [ ]` で。

## 現在のフェーズ
**発散**（セクション候補の洗い出し中）

## 次のマイルストーン
- [ ] 発散を一区切り
- [ ] 候補を部／章にグルーピング（複数の目次案）
- [ ] 目次案を1つ選定
- [ ] sections/ に空MDをscaffold
- [ ] 用語対応表のファクトチェック
